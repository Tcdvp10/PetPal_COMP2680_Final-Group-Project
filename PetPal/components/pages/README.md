<-----PET PAL----->
A clean and simple dog-care information website

Project Description
Pet-Pal is a informational, multi-page website designed to help dog owners access clear and organized information about dog breeds, care routines, training basics, and pet accessories. The site brings together content that is often scattered across multiple sources into one easy-to-navigate platform.

The project includes:
- A homepage
- Breeds directory
- Individual breed information pages
- Care & training guides
- A simple UI-based pet store
- An about page
